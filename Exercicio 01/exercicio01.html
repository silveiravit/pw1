Passo a passo de como mandar arquivos para o Github

Abrir o Git Bash

Digitar: git config --global user.name "nome de usuário"

Digitar: git config --global user.email seu email

Após esse passo você terá que pegar seu link no repositório

Digitar: git clone + seu link(esse código irá clonar o seu repositório do git)

Iremos acessar a pasta utilizando o comando cd + nome da pasta

Digitaremos: ls (para saber oq tem na pasta)

Colocaremos o arquivo na pasta que clonamos do nosso repositório

Digitaremos: git add . (para colocarmos os arquivos que estão na pasta)

Digitamos: git status (para saber oq foi feito)

Digitamos: git commit -m "Descrição do que fizemos"

E por último: git push (para mandarmos o arquivo para o github)
